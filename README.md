# Claude Code Basic — 우아한형제들 임원진 입문 교육

우아한형제들 임원진 대상 90분 Claude Code 입문 교육 자료 저장소.

## 핵심 메시지

> **바이브 코딩 → 에이전틱 코딩 → 하네스 엔지니어링**
>
> 임원진의 레버는 코드가 아니라, 조직이 AI와 일하는 방식을 설계하는 것이다.

## 구성

- [site/](site/) — Nextra 기반 교육 사이트 (세션 슬라이드 + 사후 레퍼런스)
- [docs/](docs/) — 커리큘럼, 퍼실리테이션 가이드, 데모 스크립트
- [sample-team/](sample-team/) — 실습·데모용 가상 팀 문서 데이터셋
- [CLAUDE.md](CLAUDE.md) — 저장소 작업 원칙이자 T3 하네스 엔지니어링 라이브 데모 소재

## 교육 개요

| 항목 | 내용 |
|---|---|
| 대상 | 우아한형제들 임원진 (Claude Code 첫 경험) |
| 시간 | 90분 단일 세션 |
| 도착점 | 본인의 언어로 3-tier를 설명하고, 팀에 "우리 CLAUDE.md 만들어봅시다"라고 지시할 수 있는 상태 |

## 빠르게 시작

```bash
cd site
npm install
npm run dev
```
