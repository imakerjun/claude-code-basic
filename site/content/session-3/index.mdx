# 세션 3 · T3 하네스 엔지니어링 (55~90분)

> **이 세션의 성공 신호**: 수강자가 "우리 조직의 CLAUDE.md에 이걸 넣으면 좋겠다" 라는 아이디어를 **본인 입으로 꺼냅니다.**

여기서 "이것이 내 레버다" 라는 깨달음이 일어나야 90분 교육의 목적이 달성됩니다.

> **"여러분은 코드를 짤 필요가 없습니다.**
> **여러분은 규칙을 설계합니다."**

## 하네스의 네 가지 재료

| 재료 | 역할 | 비유 |
|---|---|---|
| **CLAUDE.md** | 항상 적용되는 조직 규칙 | 사내 가이드 문서 |
| **Skills** | 반복되는 절차에 이름 붙이기 | 표준 운영 절차(SOP) |
| **Agents** | 역할 기반 위임 | 팀 내 전문 담당자 |
| **Hooks** | 정책의 자동 집행 | 게이트 키퍼 |

오늘 세션에서는 **CLAUDE.md와 Skills** 두 가지를 중심으로 다룹니다. Agents와 Hooks는 개념만 소개합니다.

## 흐름

1. CLAUDE.md 라이브 데모 (10분)
2. Skills 라이브 데모 (5분)
3. Agents · Hooks 개념 소개 (5분)
4. 도입 시 고려사항 (10분)
