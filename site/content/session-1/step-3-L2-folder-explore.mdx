# 3. L2 · 폴더 탐색 (10분)

## 프롬프트

```
sample-team/ 폴더 전체를 훑어보고 이 팀이 어떤 팀이고 무엇을 하는지 설명해줘.
```

## Claude Code의 반응

터미널에 도구 사용 로그가 순차적으로 표시됩니다.

- `Glob` — 폴더 구조 파악
- `Read` — 여러 파일을 차례로 열람
- 종합한 설명을 출력

## 이 장면에서 강조할 것

**"아무도 어떤 파일을 열라고 말하지 않았습니다."** Claude Code가 스스로 폴더 구조를 살피고, 어떤 파일을 읽어야 할지 판단하고, 순서대로 열어봅니다.

이것이 "대화형 AI"와 "에이전트형 AI"의 첫 번째 가시적 차이입니다.

## 강사 멘트 포인트

> **"여러분, 지금 터미널 왼쪽에 뜬 이 로그 보이시죠?
>
> Glob, Read, Read, Read…
>
> 이게 Claude가 스스로 한 행동입니다.
> 저는 '폴더 전체 보고 말해줘' 라고만 했습니다.
> 어떤 파일을 열지는 Claude가 정했습니다."**

## 수강자가 느껴야 하는 것

- "어? 자기가 알아서 파일을 열었네"
- "사람이 폴더 탐색하고 요약하는 거랑 비슷하네"
- "근데 30초 걸렸네"

이 세 생각이 머릿속에 뜨면 이 세션은 성공입니다.

## 예상 출력 (참고용)

```
이 팀은 결제 플랫폼 팀으로 보입니다. 주요 내용은 다음과 같습니다.

- 2026 Q2 OKR: 온보딩 자동화, 결제 오류 인지 시간 단축, 파트너사 API 리스크 관리
- 최근 회의: Q1 회고와 Q2 목표 확정, 신규 입사자(정개발) 합류
- 현재 이슈: 결제 오류 severity P0 기준에 대한 팀 내 미해결 논쟁 (5% vs 2%)

팀 구성은 김팀장, 이대리, 박주임, 최매니저, 정개발(신규) 5명입니다.
```
