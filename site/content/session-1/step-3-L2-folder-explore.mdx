# 3. L2 · 폴더 탐색

## 프롬프트

```
sample-team/ 폴더 전체를 훑어보고 이 팀이 어떤 팀이고 무엇을 하는지 설명해줘.
```

## 관찰 포인트

터미널에 도구 사용 로그가 순차적으로 표시됩니다.

- `Glob` — 폴더 구조 파악
- `Read` — 여러 파일을 차례로 열람
- 종합한 설명을 출력

## 핵심 개념

어떤 파일을 열지 사람이 지시하지 않았는데 Claude Code가 스스로 판단해 순서대로 열어봅니다. 이것이 **대화형 AI와 에이전트형 AI의 첫 번째 가시적 차이**입니다.

## 예상 출력 (참고용)

```
이 팀은 결제플랫폼팀으로 보입니다.

- 2026 Q2 OKR: 사장님광장 온보딩 자동화, 배민페이 이상 인지 시간 단축, PG 3사 호환성 리스크 관리
- 최근 회의: Q1 회고와 Q2 목표 확정, 신규 입사자(정개발) 합류
- 현재 이슈: P0 severity 기준에 대한 미해결 논쟁 (2% vs 5%)

구성원: 김팀장, 이대리, 박주임, 최매니저, 정개발(신규) 5명.
```
