# 세션 1 · T1 체험 (0~25분)

> **이 세션의 성공 신호**: 수강자가 Claude Code가 여러 파일을 스스로 열어보는 장면에서 "어?" 하는 반응을 보입니다.

## 흐름

1. 오프닝 (10분) — 3-tier 프레임 선언
2. L1 데모 (5분) — 단일 파일 요약
3. L2 데모 (10분) — 폴더 전체 탐색

이 세션의 목표는 "와우"가 아니라, **"이것이 ChatGPT와 어떻게 다른가"** 라는 질문을 수강자 머릿속에 심는 것입니다. 차이의 첫 번째 증거는 **Claude Code가 스스로 파일을 열어본다** 입니다. 이 한 장면에서 "대화형 AI"와 "에이전트형 AI"의 개념 차이가 시작됩니다.
