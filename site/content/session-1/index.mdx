# 세션 1 · 바이브 대비 + 에이전틱 첫 체감

## 흐름

1. 오프닝 — 2-모드 프레임 소개 (Vibe / Agentic + Harness)
2. L1 데모 — 단일 파일 요약
3. L2 데모 — 폴더 전체 탐색

이 세션의 목표는 **바이브 코딩(2분 HTML 데모)** 과 **에이전틱 코딩(L1/L2)** 의 첫 대비를 수강자의 눈앞에 놓는 것입니다. L2에서 Claude Code가 지시 없이 Glob·Read를 스스로 호출하는 장면이 *행위 양식으로서의 agentic*을 드러내는 첫 증거입니다.
