# 월요일 체크리스트

> **오늘 90분 교육에서 단 한 페이지만 북마크한다면, 이 페이지입니다.**

교육 다음 주 월요일 오전, 이 순서대로 해보세요. 전부 해도 30분이면 됩니다.

## 오전 10:00 — Claude Code 한 번 열기

- [ ] 본인 노트북에서 `claude` 실행
- [ ] 본인 업무 관련 폴더 하나를 대상으로 L2 프롬프트 실행

```
이 폴더를 훑어보고 주요 내용을 알려줘.
```

**목표**: 본인 손으로 성공 경험 1회.

## 오전 10:15 — 어제의 놀라운 장면 되새기기

- [ ] 세션에서 본 L3 "미처리 액션 아이템 추출" 장면을 떠올린다
- [ ] 우리 팀 회의록 3개에 같은 프롬프트를 적용해본다

```
최근 회의록 3개를 모두 읽고, 아직 처리되지 않은 액션 아이템만 담당자별로 정리해줘. 완료된 것은 제외해줘.
```

**목표**: "이거 정말 내 업무에 쓸 수 있구나" 를 본인 언어로 확인.

## 오전 10:30 — CLAUDE.md 한 줄 초안 쓰기

- [ ] 메모장이든 노션이든 어디든, 우리 본부·팀의 CLAUDE.md 초안 **첫 한 줄** 을 적는다

**규칙**: 완벽한 문서를 만들 필요 없습니다. **한 줄만** 적습니다.

예시:
- "고객 데이터 분석은 반드시 익명화부터 한다"
- "기획 문서의 첫 줄은 '왜' 로 시작한다"
- "수치에는 반드시 출처를 명시한다"

**목표**: 조직 규칙 설계의 첫 한 걸음.

## 오전 10:45 — 팀장(또는 팀원)에게 공유

> "오늘 Claude Code 교육을 들었는데, 이 한 줄을 우리 팀 CLAUDE.md로 시작하면 어떨까 싶어요. 어떻게 생각하세요?"

**목표**: 본인의 레버를 조직으로 확장하는 첫 대화.
