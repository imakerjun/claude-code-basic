# Claude Code 설치

세션 전에 참가자가 본인 노트북에 Claude Code를 설치하고 로그인까지 완료해두면 실습 15분이 매끄럽게 흘러갑니다.

## 설치

### macOS / Linux

터미널에서 다음을 실행합니다.

```bash
npm install -g @anthropic-ai/claude-code
```

### Windows

WSL 환경에서 동일한 명령을 실행합니다.

## 로그인

설치 후 터미널에서 다음을 실행합니다.

```bash
claude
```

브라우저가 열리고 Anthropic 계정으로 로그인하는 플로우가 시작됩니다. 회사 계정 정책이 있다면 정책에 맞는 계정을 사용합니다.

## 첫 실행 확인

설치와 로그인이 끝났다면 임의의 디렉터리에서 `claude`를 실행하고 간단한 질문을 던져봅니다.

```
이 폴더에 뭐가 들어 있어?
```

응답이 돌아오면 준비 완료입니다.

## 세션 당일 실습 환경

- 작업 디렉터리는 `claude-code-basic/` 또는 `claude-code-basic/sample-team/`
- 세션 전에 해당 폴더를 미리 git clone 하거나 zip으로 배포받습니다
- 네트워크 접근이 가능한지 사전 확인합니다

## 트러블슈팅

| 증상 | 원인 | 대응 |
|---|---|---|
| `claude` 명령을 찾을 수 없음 | PATH 문제 | 터미널 재시작 또는 `npm bin -g` 경로 확인 |
| 로그인 페이지가 안 열림 | 회사 방화벽 | IT에 `anthropic.com` 도메인 허용 요청 |
| 폴더를 못 읽음 | 권한 문제 | `cd`로 해당 폴더에 들어간 상태에서 `claude` 재실행 |
