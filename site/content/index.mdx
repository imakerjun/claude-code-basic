# Claude Code Basic

**우아한형제들 임원진 대상 90분 Claude Code 입문 교육**

> 바이브 코딩 → 에이전틱 코딩 → **하네스 엔지니어링**
>
> 임원진의 레버는 코드가 아니라, 조직이 AI와 일하는 방식을 설계하는 것입니다.

## 오늘의 도착점

이 90분이 끝나면 여러분은 다음 세 가지를 할 수 있게 됩니다.

1. **설명한다** — 바이브 코딩 / 에이전틱 코딩 / 하네스 엔지니어링의 차이를 본인의 언어로.
2. **판단한다** — 본인 업무에서 Claude Code를 쓸 장면을 구체적으로 지목한다.
3. **지시한다** — 팀에 "우리 CLAUDE.md를 만들어봅시다" 라고 말할 수 있다.

## 90분 구성

| 시간 | 섹션 | Tier |
|---|---|---|
| 0~10 | Opening: 3-tier 프레임 | 전체 |
| 10~25 | T1 체험 (문서 분석 L1~L2) | T1 |
| 25~40 | T2 이해 (문서 분석 L3~L5) | T2 |
| 40~55 | 실습 | T1→T2 |
| 55~75 | **T3 하네스 엔지니어링** | **T3** |
| 75~85 | 도입 시 고려사항 | 조직 |
| 85~90 | Action Planning | — |
