# 세션 2 · 에이전틱 루프 심화

## 흐름

1. L3 데모 — 여러 문서 가로질러 판단
2. L4 데모 — 계획 수립
3. L5 데모 — 실행 루프 (쓰기까지)
4. 실습 — 본인 노트북에서 재현 + 페어 회고

세션 1에서 본 agentic 루프가 **다문서 교차 → 계획 수립 → 파일 쓰기**까지 확장되는 구간입니다. 읽기만 하던 루프가 **상태를 바꾸는 루프**로 넘어가는 지점이 L5입니다.

## 세션 3과의 관계

세션 2(Agentic)와 세션 3(Harness)은 **같은 층위의 두 면**입니다.

- 세션 2는 *루프가 어떻게 움직이는지*를 체감하는 면
- 세션 3은 *그 루프가 어떤 환경·규칙 안에서 돌지*를 설계하는 면

세션 2의 마지막에 자연스럽게 떠오르는 질문 — **"Claude가 어떤 기준으로 우선순위를 정했지?"** — 이 질문의 답이 세션 3입니다.
