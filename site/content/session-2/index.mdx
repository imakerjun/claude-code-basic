# 세션 2 · T2 에이전틱 (25~55분)

> **이 세션의 성공 신호**: L3에서 수강자가 노트 필기를 시작합니다. L4에서 "작업 쪼개기를 AI가 한다고?" 반응이 나옵니다.

## 흐름

1. L3 데모 (6분) — 여러 문서 가로질러 판단
2. L4 데모 (5분) — 계획 수립
3. L5 데모 (4분) — 실행 루프
4. 실습 (15분) — 본인 노트북에서 L2 수준 재현 + 페어 회고

T1과 T2의 경계는 **"계획을 스스로 세우는가"** 입니다.

- **T1**: 질문 → 답변 (한 왕복)
- **T2**: 질문 → **계획** → 실행 → 검증 → 결과 (자기 수정 루프)

이 경계가 L4 데모에서 수강자 눈앞에 드러납니다.
