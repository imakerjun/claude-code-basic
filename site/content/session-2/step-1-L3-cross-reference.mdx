# 1. L3 · 여러 문서 가로지르기 (6분)

**이 세션의 결정적 "아하" 포인트입니다.** 여기서 반응이 없으면 뒤 섹션이 무너집니다.

## 프롬프트

```
sample-team/회의록/ 폴더의 회의록 3개를 모두 읽고,
아직 처리되지 않은 액션 아이템만 담당자별로 정리해줘.
완료된 것은 제외해줘.
```

## 왜 이 프롬프트인가

임원진이 **매주 하고 싶지만 아무도 안 해주는 일** 이기 때문입니다.

- 여러 회의록을 가로지르기
- 완료/미완료 구분
- 담당자별 재분류

각각은 간단하지만, 이 셋을 **사람이 같이 하면 30분** 걸립니다. Claude Code는 30초에 끝냅니다.

## 예상 출력

```
미처리 액션 아이템 (담당자별)

이대리
- [2026-03-29] 신규 입사자 온보딩 가이드 공유 (4/1까지)
- [2026-03-22] Q2 OKR 80건으로 수정 반영 (3/25까지)

최매니저
- [2026-03-29] severity P0 기준 재검토 후 재공유 (4/5까지)
- [2026-03-22] 결제 오류 severity 정의서 작성 (3/29까지)

정개발
- [2026-03-29] 온보딩 자동화 현황 파악 (4/5까지)
```

## 강사 멘트

> "이것이 매주 30분 걸리던 일입니다. 방금 30초에 끝났습니다.
>
> 그리고 이 작업, 여러분이 **임원 회의 들어가기 직전에 하고 싶으셨던 거 아닌가요?**"
