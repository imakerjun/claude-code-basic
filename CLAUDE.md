# CLAUDE.md

이 저장소는 Claude Code 90분 입문 교육 자료입니다.

## 작업 원칙

- 모든 교육 자료는 **설계자 관점**에서 작성합니다. "코드를 짜라"가 아닌 "시스템을 설계하라" 톤을 유지합니다.
- 한국어로 작성합니다. 외래어는 필요 시 병기합니다.
- 근거 없는 주장은 피합니다. 근거가 있을 때는 출처를 함께 적습니다.
- 과장 표현을 금지합니다. "딸깍", "한 번에", "혁명적" 같은 표현을 사용하지 않습니다.

## 문체

- 세션 슬라이드(학습 자료): `~합니다`체
- 데모 스크립트·체크리스트: `~한다`체

## 참고

이 파일 자체가 하네스 엔지니어링 섹션의 라이브 데모 소재입니다.
세션 중 이 파일에 한 줄 추가하면 Claude Code의 다음 응답이 어떻게 바뀌는지 실시간으로 보여줍니다.
